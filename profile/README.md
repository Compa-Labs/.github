# Compa Labs

A USC-affiliated student/research lab building **Compa** — a social meetup platform that uses smart matching to connect students for in-person dinners and gatherings.

## About

Compa Labs is a research organization founded by USC students Daniel Turner-Lloveras (turnerll@usc.edu) and Dillon Vuong. We build software at the intersection of social network science, real-time matching algorithms, and student community engagement.

## Active Projects

- **[Compa](https://bridge-rose-theta.vercel.app/)** — Next.js 15 + TypeScript, Supabase backend, Clerk auth. Live deployment at `bridge-rose-theta.vercel.app`.
